# Jewel 3D Scene

## Fichiers
- `index.html`
- `style.css`
- `script.js`
- `assets/hdr/studio.hdr`

## HDRI
Télécharge une HDRI studio gratuite et renomme-la `studio.hdr`, puis place-la dans `assets/hdr/`.

Sources possibles :
- https://polyhaven.com/hdris/studio
- https://www.hdri-hub.com/hdrishop/freesamples/freehdri
- https://www.lightmap.co.uk/hdrlightstudio/freestudiohdrimaps/

## Lancement
Ouvre le projet via GitHub Pages ou un serveur local.
